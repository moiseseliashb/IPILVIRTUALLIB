# 📚 IPILVirtualLIB

**Biblioteca Virtual Escolar** — Plataforma web de gestão e consulta do acervo bibliográfico escolar.

## Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** PHP 8.x (API REST) + JWT
- **Base de dados:** MySQL 8

## Perfis de utilizador
- **Aluno** — Pesquisa, requisita e lê livros online
- **Professor** — Idem + recomendações
- **Admin** — Gestão completa + dashboard de estatísticas

## Estrutura
```
IPILVirtualLIB/
├── frontend/        # Aplicação React
├── backend/         # API PHP REST
└── docs/            # Documentação e diagramas
```
